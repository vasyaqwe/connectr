# [Connectr](https://connectr.netlify.app/)

## A MERN social media app.

## Tech Stack

-   **Framework:** Vite + React + Typescript
-   **Styling:** Tailwind CSS
-   **Database:** MongoDB
-   **Server:** Express.js
-   **Image upload:** Cloudinary
-   **Schema validation:** Zod
